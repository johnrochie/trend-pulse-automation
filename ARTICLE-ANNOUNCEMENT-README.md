# 📢 Trend Pulse Article Announcement System

Automatically post new articles to social media when they go live.

## 🚀 Quick Start

```bash
# 1. Setup environment
./setup-article-announcer.sh

# 2. Generate social media posts (manual review)
node post-new-articles.js

# 3. Check generated posts
ls -la output/social-posts/
```

## 🤖 Automation Options

### Option A: Manual Posting (Easiest)
- Run `node post-new-articles.js` 
- Posts are saved as JSON files
- Copy/paste to social media manually
- Good for starting out

### Option B: Telegram Bot (Recommended)
1. Create bot with @BotFather
2. Create Telegram channel
3. Add bot as channel admin
4. Add credentials to `.env`:
   ```
   TELEGRAM_BOT_TOKEN=your_token_here
   TELEGRAM_CHANNEL_ID=@your_channel
   ```
5. Run: `node post-to-telegram.js`

### Option C: Twitter/X Automation
1. Get API keys from developer.twitter.com
2. Add to `.env`:
   ```
   TWITTER_API_KEY=your_key
   TWITTER_API_SECRET=your_secret
   TWITTER_ACCESS_TOKEN=your_token
   TWITTER_ACCESS_SECRET=your_token_secret
   DEMO_MODE=false
   ```
3. Run: `node post-to-twitter.js`

### Option D: Full Automation (Cron)
```bash
# Add to crontab (runs every 2 hours)
0 */2 * * * cd /path/to/trend-pulse-automation && node post-new-articles.js >> automation.log 2>&1

# For Telegram (every 4 hours)
0 */4 * * * cd /path/to/trend-pulse-automation && node post-to-telegram.js >> telegram.log 2>&1
```

## 🔧 Configuration

### .env File
```env
# Required
SITE_URL=https://www.trendpulse.life

# Telegram (optional)
TELEGRAM_BOT_TOKEN=your_token_here
TELEGRAM_CHANNEL_ID=@your_channel

# Twitter/X (optional)
TWITTER_API_KEY=your_key_here
TWITTER_API_SECRET=your_secret_here
TWITTER_ACCESS_TOKEN=your_token_here
TWITTER_ACCESS_SECRET=your_token_secret_here
DEMO_MODE=true  # Set to false for real posting
```

## 🚨 Troubleshooting

### "No new articles found"
- Articles must be published in last 24 hours (configurable)
- Check `publishedAtSite` field in articles.json
- Verify articles are being generated by automation

### Telegram bot not posting
- Verify bot token and channel ID
- Bot must be admin in channel
- Check Telegram API status

### Twitter API errors
- Verify all 4 API keys are correct
- Check Twitter Developer Portal for app permissions
- Ensure DEMO_MODE=false in .env
